# Candeflix

**Autora:** Candela Hernandez Fano

## Descripció

Candeflix és una aplicació web que funciona com a buscador de pel·lícules. Permet cercar pel·lícules i mostrar informació rellevant com ara una breu sinopsi, el repartiment, el director i altres detalls destacats.

## Stack Tècnic

* Angular 18
* SCSS
* Standalone Components
* Node.js
* npm
